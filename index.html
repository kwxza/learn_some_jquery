Welcome!

Welcome to my jQuery guide!

I'm kpsiDev and I created this guide as both a learning exercise for myself and a project for the FreeCodeCamp curriculum.

As a self-taught (and still learning) novice web developer, I use the internet to help me decide which of the many web technologies are worth taking the time to learn. In regards to jQuery, I've noticed two interesting things:

  1. A lot of the advice out there about jQuery says you don't need it anymore, yet
  2. the more I look at real-life JavaScript code, the more I see jQuery.

It's better to know something and not need it, right? Another tool in the toolbox.
So with this guide, I aim to help new devs like myself gain a general understanding of what jQuery is and how to use it.

Don't malign the $ sign!
jQuery ain't that scary!

I hope you learn as much from reading this guide as I learned from making it.

Before Starting

This guide assumes that the reader has a working knowledge of:

  • JavaScript
  • CSS
  • HTML
  • The Document Object Model (DOM)

Familiarity with CSS selectors and JavaScript syntax is important, as well as a general understanding of what the DOM is and how it works.

Mozilla Developer Network is a great resource to use for exploring more about all of these topics.

What is jQuery?

jQuery is a lightweight JavaScript library designed for manipulating the DOM (Document Object Model) of an HTML document. It is free and open-source under the MIT License and is the most widely-used JavaScript library, deployed by 73% of the 10 million most popular websites.

jQuery makes selecting and editing DOM elements easier for developers compared to vanilla JavaScript. The library is built for cross-browser consistency, eliminating errors caused by differing browser JavaScript engines. jQuery also simplifies event-handling, CSS animation and asychronous use of AJAX.
